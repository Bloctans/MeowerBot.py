<h1> Post object </h1>
</p>

## Attributes

- ctx: a [mb.context.CTX](./ctx.md) object 

- data: the post string
- user: [User](./user.md) 
	The post author.

- chat: str
	The ID of the chat where the post was made

- type: int

- date: datetime.datetime object
	The exact date and time at which the post was created.
